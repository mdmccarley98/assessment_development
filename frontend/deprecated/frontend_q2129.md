## Stem

## Code Snippet (optional)

## Answer Key
| Action                            | HTTP Verb     |
| --------------------------------- | ------------- |
| Create a new resource             | PUT or POST   |
| Append to an existing resource    | POST          |
| Retrieve a resource               | GET           |
| Delete a resource                 | DELETE        |
