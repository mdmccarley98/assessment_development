## Claim

## Claim Behavior (evidence)
> Spinners provide a quick way to select one value from a set. In the default state, a spinner shows its currently selected value. Touching the spinner displays a dropdown menu with all other available values, from which the user can select a new one.
> - https://developer.android.com/guide/topics/ui/controls/spinner

## Distractors
### 1.
A widget that indicates the app is loading something.


### 2.

### 3.
A widget that lets you select a valid date or time.


## Common errors, misconceptions, or irrelevant information:

A spinner is most like the "select" tag in HTML, "list box" in Win32, or a "combo box" in Mac AppKit.

1. this is probably an indeterminate progress bar

2. this is a determinate progress bar

3. this is a picker
