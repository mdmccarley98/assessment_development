## Stem
What would be the result of running the following code?

## Code Snippet (optional)
```swift
class A {
    func printMessage() {
        print("A")
    }
}

class B: A {
    override func printMessage() {
        print("B")
    }
}

let a : A = B()
a.printMessage()
```

## Answer Key
It prints B.

## Distractors

### 1.
It fails to compile because the assignment “let a : A = B()” needs explicit type casting.

### 2.
It prints A.

### 3.
It prints A, B.
